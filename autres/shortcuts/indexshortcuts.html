Les Basiques :

Mac

  • Menu Ficher Alt + FCtrl + F2 + F
  • Menu Edition Alt + ECtrl + F2 + E
  • Ouvrir nouveau doc Ctrl + NCmd + N
  • Ouvrir un Fichier Ctrl + OCmd + O
  • Rechercher Ctrl + FCmd + F (texte)
  • Tout sélectioner Ctrl + ACmd + A
  • Couper text Ctrl + XCmd + X
  • Copier text Ctrl + CCmd + C
  • Coller text Ctrl + VCmd + V
  • Annuler (undo) Ctrl + ZCmd + Z
  • Annuler "undo" (redo) Ctrl + YCmd + Y
  • Enregistrer Ctrl + SCmd + S
  • Enregistrer sous... shift + Ctrl + S

Pour widows, la touche [Alt] permet de naviguer dans la barre de menu en haut : lorsque vous appuyez sur la touche [Alt] cela active la navigation au clavier et un petit trait apparaît sous la lettre de chaque menu de cette barre, soit vous utilisez les flèches droit et gauche du clavier, soit la touche correspondante a la lettre du menu pour le déployer.

Exemple : [alt] puis [f] ouvrira le menu "file".
