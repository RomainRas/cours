Cours sur la structure de base HTML

Les éléments fondamentaux d'une page HTML

Une page HTML est composée de plusieurs éléments, chacun ayant un rôle précis. Voici une description des éléments clés que nous avons vus jusqu'ici :

  • <!DOCTYPE html> : Indique que le document est en HTML5.
  • <html lang="fr"> : Racine du document, avec l'attribut "lang" pour spécifier la langue du document.
  • <head> : Contient les métadonnées, comme le titre, l'encodage, et les liens vers les fichiers externes.
  • <body> : Contient tout le contenu visible de la page.

Autres éléments importants

En plus des éléments de base, voici d'autres éléments fréquemment utilisés dans un document HTML :

  • <header> : Section contenant les en-têtes de la page (titres, navigation, etc.).
  • <main> : Section principale de la page, unique et centrée sur le contenu spécifique de la page.
  • <section> : Permet de structurer le contenu en différentes parties logiques.
  • <footer> : Section du bas de page, souvent utilisée pour les informations de contact ou les liens supplémentaires.
  • <link> : Permet de lier des fichiers externes, comme des feuilles de style CSS ou des polices.
