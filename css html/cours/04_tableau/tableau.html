Cours sur les tableaux en HTML

Un tableau en HTML est une structure qui permet de représenter des données sous forme de grille. Il est principalement constitué des balises <table>, <tr>, <th> et <td>.

Voici la structure de base :

  • <table> : La balise englobante qui contient tout le tableau.
  • <tr> : Chaque ligne du tableau.
  • <th> : Les cellules d'en-tête (header), qui définissent les titres des colonnes ou des lignes.
  • <td> : Les cellules de données.

Exemple de tableau simple :

Nom Âge Ville
John 28 Paris
Marie 24 Lyon
Paul 32 Toulouse

Pour améliorer l'accessibilité et la compréhension du contenu d'un tableau, on peut ajouter une légende avec la balise <caption> pour décrire le tableau.
