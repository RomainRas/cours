Pour créer un tableau, il faut un ensemble de balises associées entre-elles : La première d'entre-elles est <table></table>

Il s'agit du conteneur du tableau qui englobe tout le reset.

Ensuite, dans un tableau simple, on utilise des lignes <tr></tr> qui contiennent elles-même des cellules d'entête et des cellules, respectivement <th></th> et <td></td>

Cette structure simple permet déjà, d'organiser ses données en forme de tableau :

TH 1 TD 1.1 TD 1.2
TH 2 TD 2.1 TD 2.2
TH 3 TD 3.1 TD 3.2

Pour ajouter une information concernant le tableau (et améliorer l'accessibilité ainsi que le référencement), on peut attribuer à ce tableau, un [caption] avant la première ligne : (à noter qu'il existe une propriété css pour mettre le [caption] en bas du tableau

Mon tableau exemple :
TH 1 TD 1.1 TD 1.2
TH 2 TD 2.1 TD 2.2
TH 3 TD 3.1 TD 3.2
