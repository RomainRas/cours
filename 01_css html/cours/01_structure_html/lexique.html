Lexique de Base : Structure HTML

Les éléments de base en HTML

  • <!DOCTYPE html> : Indique que le document utilise HTML5.
  • <html lang="fr"> : Élément racine du document avec l'attribut "lang" pour spécifier la langue du contenu.
  • <head> : Contient les métadonnées du document, comme l'encodage et les liens vers des ressources externes.
  • <body> : Contient tout le contenu visible de la page HTML.

Les éléments structurels en HTML

  • <header> : Utilisé pour regrouper les éléments d'en-tête de la page, comme le logo et le menu de navigation.
  • <main> : Section principale du contenu de la page, unique pour chaque page.
  • <section> : Utilisé pour structurer le contenu en parties distinctes.
  • <footer> : Contient le contenu de bas de page, comme les informations de contact et les droits d'auteur.
  • <link> : Utilisé pour lier des fichiers externes, comme des feuilles de style CSS.
