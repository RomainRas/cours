Cours Web : Les images, la gestion de projet, et le responsive design

Les différents types d'images utilisables

Voici les types d'images que vous pouvez utiliser sur le web :

SVG (Scalable Vector Graphics)

  • Type : Vectoriel
  • Utilisation : Graphiques, logos, icônes
  • Qualité : Toujours nette, même en zoomant
  • Taille de fichier : Variable, souvent petite pour des graphiques simples
  • Édition : Facile à modifier
  • Transparence : Supportée
  • Animations : Possible

JPEG (Joint Photographic Experts Group)

  • Type : Bitmap
  • Utilisation : Photographies
  • Qualité : Perte de qualité avec la compression
  • Taille de fichier : Plus petite grâce à la compression
  • Transparence : Non supportée

PNG (Portable Network Graphics)

  • Type : Bitmap avec compression sans perte
  • Utilisation : Idéal pour les images nécessitant de la transparence
  • Qualité : Maintenue, quelle que soit la compression
  • Transparence : Supportée (canal alpha)

Étapes de la création de projet et gestion client

Délais de projet

Durée estimée : 15 jours de travail effectifs. Il est important de bien quantifier chaque journée de travail.

Rencontre avec les parties prenantes

Organiser des réunions pour recueillir les besoins et attentes des utilisateurs, des équipes internes et des partenaires.

Création de l'arborescence du site

Il est essentiel de structurer le contenu du site de manière logique et intuitive pour faciliter la navigation des utilisateurs.

Conception UX

L'objectif est de créer une interface simple à utiliser, garantissant une bonne expérience utilisateur.

Concepts techniques en web design

Gestion des chemins d'accès

Il est important de différencier les chemins absolus des chemins relatifs :

  • Chemin absolu : URL complète, par exemple : Page 2
  • Chemin relatif : Utilisé pour les liens internes, par exemple : Page 2

Optimisation des unités de mesure pour le responsive design

Utilisez des unités comme %, em, rem, vh, vw pour garantir que le design s'adapte aux différents types d'écrans.

Glossaire des unités de mesure

  • px : Pixel, unité absolue
  • % : Unité relative à l'élément parent
  • em : Unité relative à la taille de l'élément parent
  • rem : Unité relative à l'élément racine (root)
  • vh : Viewport height, 1vh = 1% de la hauteur de la fenêtre
  • vw : Viewport width, 1vw = 1% de la largeur de la fenêtre
