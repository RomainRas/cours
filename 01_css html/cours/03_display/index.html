Cours sur les différents types de display en CSS

La propriété display en CSS contrôle la façon dont un élément HTML est affiché dans la page. Chaque type de display affecte le comportement de l'élément et la façon dont il interagit avec les autres éléments. Voici les principaux types de display et leur comportement.

Display: block

Les éléments avec display: block occupent toute la largeur disponible, chaque élément commence sur une nouvelle ligne.

Élément 2 (block)
Élément 3 (block)

Display: inline

Les éléments avec display: inline n'interrompent pas le flux du texte et ne commencent pas sur une nouvelle ligne.

Élément 1 (inline) Élément 2 (inline) Élément 3 (inline)

Display: inline-block

Les éléments avec display: inline-block sont similaires aux éléments inline, mais ils respectent la hauteur et la largeur définies.

Élément 1 (inline-block) Élément 2 (inline-block) Élément 3 (inline-block)

Display: none

Les éléments avec display: none sont complètement retirés du flux de la page, ils ne prennent aucune place visible.

Cet élément est masqué (display: none)

Display: grid

Les éléments avec display: grid permettent de créer une mise en page en grille, avec des lignes et des colonnes.
